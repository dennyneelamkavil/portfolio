## 🛠️ Built With

- **Next.js** - React framework for building modern web applications
- **React** - Component-based UI library
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animations and transitions
- **shadcn/ui** - Reusable UI components
- **Lucide React** - Icon library
- **Vercel** - Deployment platform

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/dennyneelamkavil/portfolio.git
```

### Navigate to the project directory

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```

in your browser to view the application.

## 📁 Project Structure

```txt
app/                  # Next.js App Router pages
components/           # Reusable UI and section components
data/                 # Portfolio datasets (projects, skills, socials, etc.)
public/               # Static assets and images
hooks/                # Custom React hooks
lib/                  # Utility functions
styles/               # Global styles
```

## 🎨 Customization

To customize this portfolio for your own use:

1. Update personal information in the `data` directory.
2. Replace images inside the `public` folder.
3. Modify theme colors in `globals.css`.
4. Update SEO metadata in `app/layout.tsx`.
5. Add or modify projects, experience, and skills datasets.

## 📈 Performance & SEO

This portfolio includes:

- SEO-friendly metadata

- Twitter card support
- Responsive images
- Optimized performance using Next.js

## 🚀 Deployment

The portfolio is optimized for deployment on Vercel.

To deploy:

1. Push your code to GitHub.
2. Import the repository into Vercel.
3. Configure environment variables (if required).
4. Deploy.

Every push to the main branch can automatically trigger a new deployment.
